# PVE-PL

Polish Proxmox site.

## Host

- Hostname: `pve`
- Hardware: GEEKOM GT1 Mega
- CPU: Intel Core Ultra 7 155H
- CPU topology: 16 cores / 22 threads shown by Linux
- RAM: 32 GB
- OS: Debian trixie
- Proxmox VE: 9.2.3
- pve-manager: 9.2.3
- Kernel: 7.0.6-2-pve
- Main LAN IP: `192.168.1.250/24`
- Default gateway: `192.168.1.1`
- WireGuard: `10.50.50.1/24`

## Main purpose

- Home Assistant OS
- Plex media stack
- Rclone sync container
- Nginx Proxy Manager
- Uptime Kuma
- Backrest
- NetBox
- Proxmox Backup Server PL
- OpenClaw / Clawbot Debian VM
